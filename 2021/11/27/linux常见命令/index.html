cd /盘符 到相应的盘符

cd 路径名称

cd ..或者- 返回上一级路径

cd ~ 返回到home目录

ll 查看当前目录下的显示的文件

ll -a 查看当前目录下含隐藏的所有文 件

xxx –version 查看当前xxx的版本

pwd 查看当前路径

pwd -p 软链接的实际路径

vim 文件名字 例如 vim hello.txt 创建一个名字叫hello的txt文

linux 编辑文件

1.进入文件 vim 文件名（vim test.txt）

2.输入 i 编辑完成之后，按ESC键 跳到命令模式（命令前面冒号是必要的）：

:w 保存文件但不退出vi

:w file 将修改另外保存到file中，不退出vi

:w! 强制保存，不推出vi

:wq 保存文件并退出vi

:wq! 强制保存文件，并退出vi

:q 不保存文件，退出vi

:q! 不保存文件，强制退出vi

:e! 放弃所有修改，从上次保存文件开始再编辑
3.退出编辑模式，在光标停留的行那边输入yy可copy当前行

在输入p则复制一行

4按esc 跳到命令模式，然后输入：wq退出并保存

cat 文件名 显示文件内容

tail -n 1 文件名 显示文件内容的最后一行

1 查看版本的命令

lsb_release -a

2查看位数（32位或64位）的命令

getconf LONG_BIT

3 查看架构（arm或amd）的命令

dpkg –print-architecture
